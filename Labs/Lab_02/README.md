# Lab 02: Introduction to PyTorch & Neural Networks

## Problem Statement
The goal of this lab was to understand the foundational mechanics of deep learning by building, training, and evaluating fundamental neural network architectures on standard datasets using PyTorch.

## Approach and Methodology
Built a custom data pipeline to handle preprocessing and batching. Constructed a standard neural network (MLP) using PyTorch, defining the forward pass, loss function, and optimization algorithms to progressively reduce training error.

## Results and Evaluation
The model was successfully trained over multiple epochs, demonstrating a clear reduction in loss and an increase in validation accuracy, proving the effectiveness of the backward pass and weight updates.

## Your Learning Outcomes
I gained a solid understanding of the PyTorch framework, particularly tensors, autograd, and the standard training loop. I learned how hyperparameters like learning rate and batch size directly impact model convergence.
